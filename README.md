# :rainbow: :rainbow::rainbow: color-randomizer :rainbow: :rainbow: :rainbow: 

Generate random colors

## Install
```
npm install color-randomizer
```

## Use
### Params
    {string} type Accepts three types: 'name', 'rgb', 'hex'

```
randomColor(); // default to name
randomColor('hex');
randomColor('name');
randomColor('rgb');
```
